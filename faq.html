F.A.Q

Cetakin itu apa?

Cetakin adalah layanan cetak online otomatis dengan notifikasi via SMS. Kami kerja sama dengan tempat cetak di fakultasmu dan kamu gausah lagi: ngantri, colok flashdisk, kena virus, dll.

Kamu cukup buka cetakin.id → cetak → terima SMS notifikasi → ambil dan bayar cetakanmu!

Lokasi Cetakin dimana aja?

Untuk mengetahui fakultas mana saja yang sudah tersedia silahkan buka dan login di app.cetakin.id!

Follow LINE@ kami disini untuk mendapat info lokasi cetak yang terbaru!

Mau pake Cetakin gimana caranya?

Untuk pengguna pertama:
Buka cetakin.id → klik “Cetak Sekarang” → login SSO → input nomor HP → Tunggu SMS Challenge Code → masukkan challenge code yang dikirim via SMS ke nomor mu dan kamu sudah siap mencetak!

Untuk pengguna yang telah verifikasi nomor HP:
Masuk ke halaman cetak → pilih lokasi, pilih pewarnaan, upload file → Cetak! Kemudian kamu akan menerima SMS ketika cetakkanmu sudah selesai → Ambil cetakkanmu di lokasi yang kamu pilih!

Setelah upload harus ngapain?

Tunggu SMS notifikasi ya! Kalau sudah menerima SMS notifikasi, kamu bisa mengambil cetakanmu

Bayarnya dimana?

Bayarnya di tempat kamu mencetak, langsung bayar ke mas-mas di sana ya :)

Cetaknya bisa kapan saja?

Benar, kamu bisa akses ke cetakin.id dan upload file kesana kapanpun dan dimanapun kamu berada. Tapi, cetakanmu akan diproses sesuai dengan jam operasional masing-masing lokasi cetak
